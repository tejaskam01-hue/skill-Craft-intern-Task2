Stopwatch Web App

A basic stopwatch built using HTML, CSS, and JavaScript.
It includes Start, Pause, Reset, and Lap functions with a clean and modern UI.
Features
Start, pause, and reset the timer
Record lap times
Millisecond accuracy

Simple and responsive design
How to Use
Open index.html in any browser and start using the stopwatch.
Files
index.html – Structure
style.css – Styling
script.js – Stopwatch logic
